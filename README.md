# Lab3-Praca-Domowa
Praca domowa nr.3 Inicjały 

Klasa MyPanel dziedziczy z klasy JPanel. Klasa posiada metode "bezier", która jako argumenty przyjmuje dwie tablice na pkty x jak i y, oraz obiekt graficzny g. 
Metoda zgodnie ze wzorem liczy krzywe bezieura, a następnie je rysuje. Następnie nadpisywana jest metoda "paint" w której uruchamiane są metody "bezieur" z odpowiednimi 
argumentami. Jedna para "x" i "y" to razem 4 pkty, 2 "zwykłe" i 2 pkty "kontrolne". W kodzie zostały użyty także gradient do zmiany koloru liter. Klasa "Statyczne" 
zawiera metody statyczne potrzebne do liczenia wzoru Beziera, czyli silnia oraz wzor newtona.
